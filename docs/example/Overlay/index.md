## 介绍

这是一个遮罩层，可往其中放入子元素。

## 属性 - Attributes & Properties

| 属性   | 说明                       | 类型    | 默认值 |
| ------ | -------------------------- | ------- | ------ |
| open   | 控制遮罩层的开关\*（必须） | boolean | false  |
| zIndex | 遮罩层的层级（非必需）     | number  | 999    |

## 方法 - Methods

| 方法   | 说明       | 参数 |
| ------ | ---------- | ---- |
| show() | 展示遮罩层 | -    |
| hide() | 隐藏遮罩层 | -    |

## 事件 - Events

| 方法    | 说明                             | Event Detail |
| ------- | -------------------------------- | ------------ |
| onClose | 点击遮罩层或按下 Escape 键的回调 | -            |

## 插槽 - Slots

| 插槽      | 说明                                   |
| --------- | -------------------------------------- |
| (default) | 可填入任意组件结构，居中在遮罩层里显示 |

## 样式变量

| 变量                        | 说明             | 默认值             |
| --------------------------- | ---------------- | ------------------ |
| --banana-overlay-background | 遮罩层的背景颜色 | rgba(0, 0, 0, 0.7) |
